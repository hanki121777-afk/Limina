# IdeaTok 다국어 처리 가이드 (i18n.md)

> 이 문서는 IdeaTok의 **6개 언어 지원**을 위한 i18n (internationalization) 처리 가이드입니다.
> 데스크톱 앱, 익스텐션, 웹 대시보드, 랜딩페이지 — 모두 동일한 규칙을 따릅니다.

---

## 1. 지원 언어 (MVP 6개)

| 언어 | 코드 | 표기 | 사용자 규모 | 비고 |
|---|---|---|---|---|
| 한국어 | `ko` | 한국어 | 5천만 명 | 본진 |
| English | `en` | English | 15억 명 | 글로벌 기본값 |
| 日本語 | `ja` | 日本語 | 1.3억 명 | 직장인/창업 시장 |
| 简体中文 | `zh-CN` | 简体中文 | 9억 명 | 중국 본토 |
| 繁體中文 | `zh-TW` | 繁體中文 | 4천만 명 | 대만, 홍콩 |
| Español | `es` | Español | 5억 명 | 스페인 + 중남미 |

**중요:** 중국어 간체(zh-CN)와 번체(zh-TW)는 **반드시 분리**. 같은 언어가 아님.

v4 추가 예정 언어: pt-BR (포르투갈어), de (독일어), fr (프랑스어), id (인도네시아어), vi (베트남어), th (태국어)

### 3-3. 초기화 코드

```typescript
// desktop-app/src/renderer/i18n/index.ts
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import ko from './locales/ko.json';
import en from './locales/en.json';
import ja from './locales/ja.json';
import zhCN from './locales/zh-CN.json';
import zhTW from './locales/zh-TW.json';
import es from './locales/es.json';
import { detectOSLocale } from './detect-os-locale';

const savedLocale = window.electronAPI.getConfig('user.language');
const defaultLocale = savedLocale || detectOSLocale() || 'en';

i18n
  .use(initReactI18next)
  .init({
    resources: {
      ko: { translation: ko },
      en: { translation: en },
      ja: { translation: ja },
      'zh-CN': { translation: zhCN },
      'zh-TW': { translation: zhTW },
      es: { translation: es },
    },
    lng: defaultLocale,
    fallbackLng: 'en',
    interpolation: { escapeValue: false }, // React가 알아서 escape
  });

// 언어 변경 시 config에 저장 + html lang 속성 업데이트
i18n.on('languageChanged', (lng) => {
  window.electronAPI.setConfig('user.language', lng);
  document.documentElement.lang = lng;
});

export default i18n;
```

### 3-4. OS 언어 자동 감지

```typescript
// desktop-app/src/renderer/i18n/detect-os-locale.ts
import { app } from 'electron';

export function detectOSLocale(): string {
  // Electron에서는 메인 프로세스에서 호출 후 IPC로 전달
  const osLocale = navigator.language; // 'ko-KR', 'en-US', 'ja-JP', etc.
  
  // 정확한 매칭
  if (osLocale === 'zh-CN') return 'zh-CN';
  if (osLocale === 'zh-TW') return 'zh-TW';
  if (osLocale === 'zh-HK') return 'zh-TW'; // 홍콩은 번체
  
  // 언어 코드만 추출 (ko-KR → ko)
  const lang = osLocale.split('-')[0];
  
  if (['ko', 'en', 'ja', 'es'].includes(lang)) return lang;
  if (lang === 'zh') return 'zh-CN'; // 기본 간체
  
  // 지원 안 하는 언어는 영어로
  return 'en';
}
```

const locales = ['ko', 'en', 'ja', 'zh-CN', 'zh-TW', 'es'];

export default getRequestConfig(async ({ locale }) => {

});

## 7. 통화/날짜/숫자 포맷

### 7-1. 통화 포맷

```typescript
// shared/utils/format.ts
export function formatCurrency(amount: number, locale: string): string {
  const currencyMap: Record<string, string> = {
    'ko': 'KRW',
    'en': 'USD',
    'ja': 'JPY',
    'zh-CN': 'CNY',
    'zh-TW': 'TWD',
    'es': 'EUR',
  };
  
  return new Intl.NumberFormat(locale, {
    style: 'currency',
    currency: currencyMap[locale] ?? 'USD',
  }).format(amount);
}

// 사용
formatCurrency(9900, 'ko');   // "₩9,900"
formatCurrency(7.99, 'en');   // "$7.99"
formatCurrency(980, 'ja');    // "¥980"
formatCurrency(45, 'zh-CN');  // "¥45.00"
formatCurrency(6.99, 'es');   // "6,99 €"
```

### 7-2. 날짜 포맷

```typescript
export function formatDate(date: Date, locale: string): string {
  return new Intl.DateTimeFormat(locale, {
    year: 'numeric',
    month: 'long',
    day: 'numeric',
  }).format(date);
}

// 사용
formatDate(new Date('2026-05-23'), 'ko');     // "2026년 5월 23일"
formatDate(new Date('2026-05-23'), 'en');     // "May 23, 2026"
formatDate(new Date('2026-05-23'), 'ja');     // "2026年5月23日"
formatDate(new Date('2026-05-23'), 'zh-CN');  // "2026年5月23日"
formatDate(new Date('2026-05-23'), 'es');     // "23 de mayo de 2026"
```

### 7-3. 상대 시간

```typescript
export function formatRelativeTime(date: Date, locale: string): string {
  const rtf = new Intl.RelativeTimeFormat(locale, { numeric: 'auto' });
  const diffMs = date.getTime() - Date.now();
  const diffDays = Math.round(diffMs / (1000 * 60 * 60 * 24));
  
  if (Math.abs(diffDays) < 1) {
    const diffHours = Math.round(diffMs / (1000 * 60 * 60));
    return rtf.format(diffHours, 'hour');
  }
  return rtf.format(diffDays, 'day');
}

// 사용
formatRelativeTime(new Date(Date.now() - 86400000), 'ko');  // "어제"
formatRelativeTime(new Date(Date.now() - 86400000), 'en');  // "yesterday"
```

### 7-4. 숫자 포맷

## 10. 번역 워크플로우

### 10-1. 기본 흐름

1. **한국어로 먼저 작성** (본진 언어)
2. **Claude API로 5개 언어 자동 번역**
3. **검수 및 수정** (특히 일본어, 중국어는 원어민 체크 추천)
4. **JSON 파일에 반영**

### 10-2. 자동 번역 스크립트

```typescript
// scripts/translate.ts
import Anthropic from '@anthropic-ai/sdk';
import fs from 'node:fs';

const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

async function translateMessages(targetLang: string) {
  const ko = JSON.parse(fs.readFileSync('messages/ko.json', 'utf-8'));
  
  const response = await client.messages.create({
    model: 'claude-sonnet-4-20250514',
    max_tokens: 8000,
    messages: [{
      role: 'user',
      content: `Translate this JSON to ${targetLang}. Keep keys identical, only translate values. Maintain JSON structure exactly.

${JSON.stringify(ko, null, 2)}`
    }]
  });
  
  const translated = JSON.parse(response.content[0].text);
  fs.writeFileSync(`messages/${targetLang}.json`, JSON.stringify(translated, null, 2));
}

['en', 'ja', 'zh-CN', 'zh-TW', 'es'].forEach(translateMessages);
```

### 10-3. 비용

500개 번역 키 기준:
- 입력 5,000 토큰 × 5개 언어 = 25,000 토큰
- 출력 5,000 토큰 × 5개 언어 = 25,000 토큰
- Sonnet 기준: 약 630원 (1회성)

새 텍스트 추가될 때마다 그것만 번역하면 됨.

### 10-4. 번역 키 누락 체크

```typescript
// scripts/check-translations.ts
const langs = ['en', 'ja', 'zh-CN', 'zh-TW', 'es'];
const ko = require('./messages/ko.json');

function flattenKeys(obj: any, prefix = ''): string[] {
  return Object.entries(obj).flatMap(([key, val]) => {
    const fullKey = prefix ? `${prefix}.${key}` : key;
    return typeof val === 'object' ? flattenKeys(val, fullKey) : [fullKey];
  });
}

const koKeys = new Set(flattenKeys(ko));

langs.forEach(lang => {
  const trans = require(`./messages/${lang}.json`);
  const transKeys = new Set(flattenKeys(trans));
  const missing = [...koKeys].filter(k => !transKeys.has(k));
  
  if (missing.length > 0) {
    console.error(`[${lang}] Missing ${missing.length} keys:`, missing);
  }
});
```

## 11. 언어별 주의사항

### 11-1. 한국어 (ko)

- "고객님" vs "당신" — IdeaTok은 친근한 "당신" 사용 (밀당 알림 톤과 일관)
- 종결어미 일관성: "~합니다" (정중) vs "~해요" (친근) — **친근 톤 통일**
- 영문 폰트 사용 금지 — Pretendard만

### 11-2. English (en)

- Sentence case 사용 ("View idea" — Title Case 아님)
- 미국 영어 기준 ("color" not "colour")
- 가격 표시: `$7.99/month` 또는 `$7.99 / mo`

### 11-3. 日本語 (ja)

- 정중체 "ですます체" 사용 (한국어 정중 톤과 동일)
- 외래어는 가타카나 (アイデア = idea)
- 띄어쓰기 없음 (단, 영문/숫자 앞뒤는 띄어쓰기)

### 11-4. 中文简体 (zh-CN)

- 본토 사용자 대상 (대륙)
- 간체자 (idea = 主意, 想法)
- 결제 통화: CNY (¥)

### 11-5. 中文繁體 (zh-TW)

- 대만/홍콩 사용자 대상
- 번체자 (idea = 主意, 想法 — 한자 차이는 거의 없지만)
- 결제 통화: TWD (NT$)
- **간체와 절대 혼용 금지** — 대만 유저는 매우 민감

### 11-6. Español (es)

- 중남미 + 스페인 모두 커버
- "tú" (친근) vs "usted" (격식) — IdeaTok은 "tú" 사용
- 가격: `6,99 €` (쉼표 소수점)
- 텍스트 길이가 영어보다 30% 정도 김 → 레이아웃 여유 필요

## 14. 주의사항

- **번역 키 추가 시 즉시 5개 언어 동기화** — 누락 시 fallback이 영어로 표시되어 어색함
- **AI 자동 번역은 검수 필수** — 특히 마케팅 문구, 법적 문서, 결제 관련
- **이모지는 모든 언어에 동일하게 적용 가능** — 안전
- **숫자 형식 차이 주의** — 1,000 (영어/한국) vs 1.000 (유럽) vs 1,000 (중국 표기)
- **언어 변경 시 페이지 리로드 금지** — `i18n.changeLanguage()` + 컴포넌트 리렌더만으로 충분
- **번역 파일은 절대 코드에 하드코딩 금지** — 모두 외부 JSON 파일로 분리
