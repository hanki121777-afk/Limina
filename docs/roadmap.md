## 🚀 v2 — 모바일 + 멀티 브라우저 (출시 후 3개월 목표)

**목표**: PC 외 환경 데이터 확보. 사용자 풀 2배 확장.

### 신규 기능

#### 1. 모바일 앱 (Capacitor 기반)
- iOS / Android 동시 출시
- 핸드폰에서 직접 텍스트 수집 (인스타, 틱톡, 모바일 유튜브 등)
- iOS는 Share Extension 방식 (Apple 정책 제약)
- Android는 Accessibility Service 활용 가능
- **개발 난이도**: 상 (3개월 작업)

#### 2. Firefox 익스텐션
- 크롬 익스텐션 코드 90% 재활용
- Manifest V3 호환 작업 약간 필요
- **개발 난이도**: 하 (1주)

#### 3. Safari 익스텐션
- Apple 개발자 계정 필요 ($99/년)
- WebExtensions API 활용
- **개발 난이도**: 중 (2~3주)

#### 4. 유튜브 자막 분석
- 현재는 제목 + 댓글만 수집
- 자막 API로 영상 내용 자체 분석
- AI 입력 토큰 증가 → API 비용 약간 상승 (월 ~50원/유저)
- **개발 난이도**: 중 (2주)

#### 5. 지메일 본문 분석
- 현재는 제목 + 미리보기만
- 본문 전체 분석 (옵트인, Pro 기능)
- 개인정보 마스킹 강화 필수
- **개발 난이도**: 중 (2주)

## 📊 v3 — 분석 고도화 (출시 후 6개월 목표)

**목표**: 아이디어 품질 + 유저 참여도 강화.

### 신규 기능

#### 1. 트렌드 키워드 (실시간 검색량)
- 대시보드 상단에 "지금 떠오르는 키워드" 표시
- Google Trends API 또는 Naver DataLab 활용
- 익명 통계 기반, 개인 데이터 노출 없음
- **개발 난이도**: 중 (2주)

#### 2. 카테고리별 알림 설정
- "IT/AI 관련 GOLD만 알림" 같은 세부 필터
- 설정 → 알림 → 카테고리별 On/Off
- **개발 난이도**: 하 (1주)

#### 3. 주간 인사이트 이메일
- 매주 월요일 아침 발송
- "지난주 당신의 관심사 흐름" 요약
- 이메일 발송: Resend.com 또는 Supabase Edge Function
- **개발 난이도**: 하 (1주)

#### 4. 실행 액션 플랜 자동 생성
- 단계별 프롬프트 5개 외에 **30일 액션 플랜** 추가
- "Week 1: 시장조사 → Week 2: 프로토타입..."
- AI 출력 토큰 증가 (월 ~30원/유저)
- **개발 난이도**: 하 (1주)

#### 5. 아이디어 진화 추적
- 같은 주제의 아이디어가 점차 구체화되는 과정 시각화
- 5월 10일 BRONZE → 5월 17일 SILVER → 5월 23일 GOLD
- 키워드 해시 유사도 기반 자동 묶기
- **개발 난이도**: 중 (2주)

## 🌐 v4 — 확장 (출시 후 12개월 목표)

**목표**: 글로벌 시장 확대 + 차세대 입력 방식.

### 신규 기능

#### 1. 추가 언어 (5개 + )
- 🇧🇷 포르투갈어 (브라질, 2.6억 명)
- 🇩🇪 독일어 (1.3억 명)
- 🇫🇷 프랑스어 (3억 명, 아프리카 포함)
- 🇮🇩 인도네시아어 (2.7억 명)
- 🇻🇳 베트남어 (1억 명)
- 🇹🇭 태국어 (7천만 명)
- **개발 난이도**: 하 (Claude API 자동 번역 활용, 1주)

#### 2. 음성 메모 분석 (Whisper API)
- 유저가 음성 메모 녹음 → Whisper로 텍스트 변환 → 분석
- 운전 중, 산책 중 떠오른 아이디어 캡처
- Whisper API 비용: 분당 $0.006
- **개발 난이도**: 중 (3주)

#### 3. 트리거 키워드 고급화
- 정규식 지원
- 카테고리별 트리거 설정
- 트리거 매치 시 즉시 분석 (현재는 옵션, 기본 OFF)
- **개발 난이도**: 하 (1주)

#### 4. 아이디어 묘지 (Graveyard)
- 점수 낮아서 소각된 아이디어들도 대시보드 "묘지" 탭에 표시
- 투명성 확보, 신뢰도 상승
- 30일 후 자동 영구 삭제
- **개발 난이도**: 하 (1주)

## ❌ 절대 안 만드는 것 (영구 제외)

### 1. 게시판/공유 기능
- 다른 유저와 아이디어 공유 → 구독 이유 사라짐
- 영감을 게시판에서 얻으면 Limina 가치 붕괴
- **Limina은 1인용 아이디어 발견 도구**

### 2. 협업 모드 / 팀 기능
- 위와 동일한 이유
- 공동창업자 매칭 같은 건 다른 서비스의 영역

### 3. 아이디어 NFT / 블록체인
- 트렌드 따라가지 말 것
- 핵심 가치와 무관

### 4. 아이디어 배틀 / 투표
- 게이미피케이션은 핵심 가치 흐림
- 유저가 "내 아이디어"에 집중하게 해야 함

### 5. 광고 모델
- Free 유저에게 광고 띄우는 거 금지
- 광고를 위해 데이터 수집한다는 오해 발생
- 수익은 오직 Pro 구독으로

### 6. 외부 도구 통합 (Notion/Slack/Trello 등)
- 아이디어 출력은 "프롬프트 복사" 한 가지만
- 통합 늘면 복잡도만 증가, 핵심 가치 흐림

## 💡 아이디어 풀 (검토 중)

아래는 검토 단계이며, 어느 버전에 들어갈지 미정.

- **Pro 유저 전용 매주 1회 보너스 분석** — 일요일 저녁 7일치 분석 무조건 실행
- **다크/라이트 모드 토글** — 현재는 다크만, 라이트 모드 요청 들어오면 추가
- **데스크톱 앱 단축키** — 유저가 단축키로 즉시 분석 트리거 (예: Ctrl+Shift+I)
- **AI 모델 선택권 (Pro 유저)** — Sonnet 외에 Opus 선택 가능 (비용 차등)
- **분석 히스토리 PDF 내보내기** — 한 달치 아이디어 모아서 PDF
- **Apple Watch / 스마트워치 알림 미러링** — 모바일 앱 출시 후 연동

이런 아이디어가 떠오르면 이 섹션에 추가해두고, 분기별로 검토.
